# Contributing

Thank you for contributing to **github_commit_status**!

Your contributions are very much welcomed and credit will be given. For comprehensive contributing guidelines, including setup instructions, code style, and pull request guidelines, please visit:

**[ReadTheDocs Online: Full Contributing Guide](https://github-commit-status.readthedocs.io/en/latest/contributing/)**

**[Repo: Full Contributing Guide](./docs/contributing.md)**

The Contribution Guide includes:
- Local development setup
- Testing instructions
- Code style guidelines
- Pull request best practices
- Code of Conduct

If you have any questions, feel free to open a [GitHub issue](https://github.com/glenjarvis/github-commit-status/issues) or reach out to the community.
